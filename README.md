# mcb-crowdsale

# contracts

**contracts/MCBCrowdsale.sol**

MCBCrowdsale is a contract to sell MCB token for MCB holders.

**contracts/MCBVesting.sol**

MCBVesting is a contract to distribute MCB tokens to beneficiaries.

# install

```shell
npm install
npx hardhat compile
```

# run tests

```shell
npx hardhat test
```

# run coverage

```shell
npx hardhat coverage
```
